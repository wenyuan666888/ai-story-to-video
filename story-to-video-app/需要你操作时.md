# 需要你操作时按下面做

当前开发服务器已启动时，应用在 **http://localhost:3000** 可访问。

---

## 配置入口（最终版）

**👉 需要接入什么、怎么配，请直接看同目录的：**  
**[接入说明.md](./接入说明.md)**

里面写清了：

- **必接**：Supabase（认证 + 数据库 + 存储）— 创建项目、复制 URL 与 anon key、执行 SQL、开邮箱登录
- **可选**：OpenAI 兼容（故事→分镜 + 图片关键词）
- **可选**：Unsplash（与描述相关的图片）
- 视频无需接入

按 **接入说明.md** 配好 `.env.local` 并重启 `npm run dev` 后，即可注册、登录、创建项目并走完整流程。
